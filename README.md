# 用户管理系统

## 📖 项目简介

基于 **Spring Boot + MySQL + Thymeleaf** 的用户管理系统，实现完整的用户增删改查功能，包含登录注册、权限控制等核心模块。

## 🚀 功能特性

### 🔐 用户认证模块
- **用户登录/注册** - 完整的身份验证流程
- **安全退出** - 安全的会话管理

### 👥 用户管理模块
- **用户列表展示** - 清晰的数据展示界面
- **用户信息管理** - 完整的增删改查（CRUD）功能
- **用户搜索** - 快速定位目标用户
- **状态管理** - 启用/禁用用户账户

### 🛡️ 权限控制模块
- **角色区分** - 管理员/普通用户权限分离
- **页面权限** - 基于角色的页面访问控制
- **操作验证** - 细粒度的操作权限校验

## 💻 技术栈

### 后端技术
- **Spring Boot 3.2.0** - 项目主框架
- **Spring MVC** - Web层框架
- **MyBatis** - 数据持久层
- **MySQL 8.0** - 数据库
- **Druid** - 数据库连接池
- **Thymeleaf** - 模板引擎

### 前端技术
- **HTML5** - 页面结构
- **JavaScript** - 交互逻辑
- **Thymeleaf** - 模板语法
